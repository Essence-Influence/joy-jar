# Joy Jar 🫙

A warm, interactive joy-tracking web app — add daily joys, pull random slips, and reflect on your patterns across the joy continuum.

## About

**Joy Jar** is a responsive, single-file web app for capturing and revisiting moments of joy. Inspired by the joy jar ritual and joy continuum framework, it helps you build a daily gratitude and awareness practice — one slip at a time.

## Features

- 🫙 Animated jar that fills as you log joys
- ✦ Five joy categories mapped to the continuum: Simple, Sensory, Relational, Creative, Transcendent
- 🎲 Random slip pull with rotating daily reflection prompts
- 🔥 Streak tracking, category breakdown chart, and daily count
- 🔍 Filter your joy collection by zone
- 📱 Fully responsive — mobile bottom nav, desktop sidebar layout
- ⚡ Zero dependencies, no server required — all data persists in localStorage

## Usage

Download `index.html` and open it in any modern browser. No build step, no install required.

```bash
git clone https://github.com/your-username/joy-jar.git
cd joy-jar
open index.html
```

## The Joy Continuum

Joy Jar is built around the Joy Continuum — a framework for understanding joy across four dimensions:

| Zone | Description |
|------|-------------|
| **Simple** | Everyday, effortless joys close at hand |
| **Sensory** | Warmth, texture, smell, taste, sound — the body as a joy instrument |
| **Relational** | Laughter, belonging, being seen — joy that requires another person |
| **Creative** | Making, playing, flow — joy that lives in the doing |
| **Transcendent** | Awe, beauty, meaning — joy that enlarges the self |
